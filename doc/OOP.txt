interface ต่างกับ abstract class 
	
	interface คือ เอาไว้ซ้อน method ที่เอาไว้ process เอาไปเขียน process แยก
	abstract class สามารถมี method ที่ไม่เป็น method ได้ แต่ class ไม่สามารถมี abstarct method ได้ มีความหยืดหยุ่นมากกว่า interface ส่วนใหญ่ใช้ภายในบริษัท หรือ ทีม
